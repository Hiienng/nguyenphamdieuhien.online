Table Definitions

1. vn_gold_24h_hist CRAWLING_BOT_DB

Description: Daily gold prices from major Vietnamese gold retailers

Data Range: 2009-08-01 to present

Source: 24h.com.vn

Column Type Description Example
date DATE (PK) Trading date 2026-01-19
type VARCHAR(50) (PK) Gold brand/type DOJI HN, SJC, PNJ HCM
buy_price NUMERIC Buy price (VND per tael) 85,000,000
sell_price NUMERIC Sell price (VND per tael) 87,000,000
Note: 1 tael (lượng) = 37.5 grams. Prices are in Vietnamese Dong (VND).

2. vn_silver_phuquy_hist CRAWLING_BOT_DB

Description: Daily silver prices from Phu Quy Jewelry

Source: phuquy.com.vn

Column Type Description Example
date DATE (PK) Trading date 2026-01-19
crawl_time TIMESTAMP (PK) Time when data was crawled 2026-01-19 08:30:00
buy_price NUMERIC Buy price (VND per tael) 1,200,000
sell_price NUMERIC Sell price (VND per tael) 1,350,000

3. vn_sbv_interbankrate CRAWLING_BOT_DB

Description: State Bank of Vietnam interbank rates and policy rates

Source: sbv.gov.vn (Official API)

Column Type Description Example
date DATE (PK) Effective date 2026-01-15
crawl_time TIMESTAMP Time when data was crawled 2026-01-19 08:30:00
ls_quadem NUMERIC Overnight rate (%/year) 4.25
ls_1w NUMERIC 1-week rate (%/year) 4.35
ls_2w NUMERIC 2-week rate (%/year) 4.45
ls_1m NUMERIC 1-month rate (%/year) 4.55
ls_3m NUMERIC 3-month rate (%/year) 4.65
ls_6m NUMERIC 6-month rate (%/year) 4.75
ls_9m NUMERIC 9-month rate (%/year) 4.85
doanhso_quadem NUMERIC Overnight volume (billion VND) 15,000
doanhso_1w ... doanhso_9m NUMERIC Volume for each term (billion VND) 5,000
rediscount_rate NUMERIC SBV rediscount rate (%/year) 3.0

Note: SBV only publishes new rates when there are policy changes, so data is not updated daily.

4. vn_bank_termdepo CRAWLING_BOT_DB

Description: Term deposit interest rates from Vietnamese commercial banks

Active banks (crawling enabled):

  • ACB - Asia Commercial Bank (acb.com.vn) - HTTP
  • CTG - VietinBank (vietinbank.vn) - HTTP
  • VCB - Vietcombank (vietcombank.com.vn) - Selenium

Disabled banks (React/Angular SPA - no public API):

  • TCB - Techcombank (techcombank.com) - React SPA
  • MBB - MB Bank (mbbank.com.vn) - AngularJS
  • VPB - VPBank (vpbank.com.vn) - React SPA
Column Type Description Example
date DATE Effective date 2026-01-19
bank_code VARCHAR Bank stock code ACB, TCB, MBB, VPB, CTG, VCB
crawl_time TIMESTAMP Time when data was crawled 2026-01-19 08:30:00
term_noterm NUMERIC No-term/demand deposit rate (%/year) 0.2
term_1m NUMERIC 1-month term rate (%/year) 2.9
term_2m NUMERIC 2-month term rate (%/year) 3.0
term_3m NUMERIC 3-month term rate (%/year) 3.4
term_6m NUMERIC 6-month term rate (%/year) 4.3
term_9m NUMERIC 9-month term rate (%/year) 4.3
term_12m NUMERIC 12-month term rate (%/year) 4.8
term_13m NUMERIC 13-month term rate (%/year) 4.85
term_18m NUMERIC 18-month term rate (%/year) 4.8
term_24m NUMERIC 24-month term rate (%/year) 5.0
term_36m NUMERIC 36-month term rate (%/year) 5.0
Note: Data is crawled daily. Some banks may not have all term options. Rates shown are for VND deposits with standard interest at maturity.

5. global_macro GLOBAL_INDICATOR_DB

Description: Global market indicators (commodities and indices)

Source: Yahoo Finance via yfinance library

Column Type Description Ticker Example
date DATE (PK) Trading date - 2026-01-17
crawl_time TIMESTAMP Time when data was crawled - 2026-01-19 08:30:00
gold_price NUMERIC COMEX Gold Futures (USD/oz) GC=F 2,715.30
silver_price NUMERIC Silver Futures (USD/oz) SI=F 31.25
nasdaq_price NUMERIC NASDAQ Composite Index ^IXIC 19,630.20
Note: Global market data reflects the previous trading day's closing prices (US market hours).
